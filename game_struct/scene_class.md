Псевдокласс

Каждая сцена это своего рода мини игра. каждая сцена получает на вход gameState, сохраняет его, перезаписывает поля gameState так, как нужно мини игре.


Задачи:
1) Создать массив процессов для каждой персоны. ( ИИ для ботов )
2) Создать массив евентов(реакций) для текущей игры. (Например если игрок подошел к персоне и при этом нажал кнопку E, то ...)
3) Задать рендерер.
4) Запустить цикл сцены.


поля:
```c
gameState* gs;
```

интерфейс:
```c
byte (*scene_init)(gameState* state);
byte (*scene_update)();  
byte (*scene_destroy)();
```

результат работы: возвращает тип byte.
